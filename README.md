Eshop

Eshop - a web application for an online store, developed with TypeScript and React.

🚀 Features

Viewing the product list

Searching and filtering products

Adding products to the cart

User authentication and registration

🛠️ Technologies

React

TypeScript

Redux Toolkit

RTK Query

React Router

Axios
